1 下载与安装

下载：

wget url #下载文件 （wget 命令常用于下载）
curl www.baidu.com #请求指定网址并显示 （curl 命令可用于各种请求方式，常用于上传与请求）

安装：解压缩 （如：tar -xzvf soft.tar.gz），进入解压后的文件夹

8 SSH

ssh [-p port] user@remote

user 是在远程机器上的用户名，如果不指定的话默认为当前用户
remote 是远程机器的地址，可以是IP/域名，或者是别名
port 是SSH Server监听的端口，如果不指定，就为默认值22
(使用exit退出当前用户的登录)

设置ssh密钥：ssh-keygen

上传公钥到服务器：

ssh-copy-id -p port user@remote（ssh-copy-id -p 22 deepin2@192.168.56.132）
让远程服务器记住密钥，之后免密登陆
